What is Support Vector Regression?

Support Vector Regression (SVR) is a supervised learning algorithm derived from Support Vector Machine (SVM). SVM is a classification algorithm used to divide data into two groups. SVR is a regression and is used to predict discrete values, typically used with time-series data and forecasting.
